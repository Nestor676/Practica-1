Elementos HTML

Para saber qué es un elemento, primero tenemos que entender la diferencia entre una etiqueta y un elemento. Las etiquetas son las partes que definen el inicio y el final de un componente en HTML; estas se delimitan con los símbolos < y >. A diferencia de esto, un elemento incluye la etiqueta de apertura, el contenido y la etiqueta de cierre. Básicamente, una etiqueta es solo el comando, por así decirlo (<p>), pero un elemento incluye todo: la etiqueta y el contenido, si lo hay.

Elementos anidados

El anidamiento es colocar un elemento dentro de otro elemento. Por ejemplo, si quieres poner una imagen que, cuando le des, te lleve a una página web distinta, deberás poner la etiqueta "a" dentro de la de "img".

Atributos HTML

Los atributos en HTML son elementos que añaden información extra a las etiquetas HTML para modificar el comportamiento o la apariencia. Estos siempre se colocan dentro de la etiqueta abierta. Algunos atributos son id, class, style...

Texto formateado

El texto formateado es cuando alguna etiqueta cambia el texto de alguna forma. Por ejemplo, "b" o "strong", lo que hacen estas etiquetas es poner el texto en negrita; la diferencia entre estos dos sería que "b" aplica negrita sin ninguna importancia y "strong" aplica importancia a esa palabra. Con "i" y "em" pasa lo mismo, pero ponen el texto en cursiva; "em" es el que le da importancia al texto. La etiqueta "u" subraya el texto indicado; "strike" y "s" aplican una línea sobre el texto, indicando que ya no es importante.
La etiqueta "sup" se utiliza para elevar un número como exponentes matemáticos: 42. Otra etiqueta es "sub", que se utiliza para poner texto en la posición de subíndice.

Comentarios en HTML

Los comentarios en HTML sirven mucho, ya sea para hacer un trabajo en grupo o también para dejar comentarios a tu yo del futuro; algo que va muy bien para no perder lo que estabas haciendo, ya que la mayoría de gente no hace todo el código sin parar. Estos comentarios también sirven para comentar un trozo de código que no funciona o está dando problemas.

Imágenes

La etiqueta "img" se utiliza para insertar imágenes en nuestra página web. No hace falta cerrarla, y es necesario poner una descripción de la imagen en "alt", ya que si no carga, saldrá esa definición. Para cambiar la altura y la anchura se usan los atributos "height" y "width". Esto es necesario para que una imagen no se vea muy grande o, por el contrario, muy pequeña.

Enlaces

La etiqueta "a" nos servirá para poner un enlace donde le indiquemos en el "href". Este enlace se puede poner dentro de un texto, imagen o icono. Tablas en HTML

La etiqueta "table" nos crea una tabla. Con "tr" creamos las filas y con "td" las celdas. Se puede modificar con los atributos "border", que añade un borde a la tabla; "cellpadding", que define el espacio entre el contenido y la celda; y "cellspacing", que controla el espacio entre celdas.
