Evolucion de HTML5

La evolucion de HTML es importante ya introduce nuevas etiquetas y funcionalidades que permiten mejorar la gestion del contenido multimedia, formularios avanzados y mejoras en la semantica de la pagina web. Algunas etiquetas nuevas son "section" se usa para agrupar informacion relacionada en una sola pagina,"article" representa un bloque auntonomo de contenido,"aside" define un contenido relacionado con el contenido principal pero no es esencial,"header" y "footer" etiquetas para definir las cabeceras y los pies de paginas,"nav" especifica las barras de navegacion, "figure" y "figcaption" agrupa elementos multimedia i sus descripciones.

Formularios en HTML

Los formularios son una herramiento muy importante, ya que permiten enviar datos al servidor, en HTML5 sus funciones an aumentado, estas mejoran su usabilidad y validacion.
Se crea con la etiqueta "form" este actua como elemento de entrada de todos los datos.

La etiqueta "input" es el elemento mas versatil para crear campos de entrada.
