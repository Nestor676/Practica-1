HTML

Es un lenguaje de marcas para crear páginas web. Este lenguaje define la estructura de la página con etiquetas que el navegador de preferencia interpreta para mostrar el contenido.

Estructura de HTML

Lo primero en una estructura de HTML es el !DOCTYPE html, para indicar al archivo de qué tipo es. Otras etiquetas básicas serían html, head y body, las cuales son esenciales para la estructura de HTML.

Etiquetas básicas

Una etiqueta esencial es title, ya que le pondrá un título a nuestra página web, que aparecerá en la pestaña del navegador.

Las etiquetas h y un número (h1, h2, h3,...) se utilizan para definir títulos y subtítulos.
Ej:

Título 1

Título 2

Título 3

Título 4

Título 5
Título 6

La etiqueta br fuerza al texto a comenzar una nueva línea. De hecho, se puede ver cómo en el texto anterior he separado el "Ej:" del texto usando esta etiqueta.

Otra sería & nbsp la cual inserta un espacio en blanco.
Hola mundo, aunque parece que he escrito separado el "Hola mundo", realmente está todo junto.

Elementos HTML

Para saber qué es un elemento, primero tenemos que entender la diferencia entre una etiqueta y un elemento. Las etiquetas son las partes que definen el inicio y el final de un componente en HTML; estas se delimitan con los símbolos < y >. A diferencia de esto, un elemento incluye la etiqueta de apertura, el contenido y la etiqueta de cierre. Básicamente, una etiqueta es solo el comando, por así decirlo (<.p>), pero un elemento incluye todo: la etiqueta y el contenido, si lo hay.

Elementos anidamiento

El anidamiento es colocar un elementos dentro de otro elemento, por ejemplo si quieres poner una imagen que cuando le des te lleve a una pagina web distinta, deberas de poner la etiqueta "a" dentro de la de img.
